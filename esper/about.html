About Esper.lol

Esper.lol

Esper.lol is a Generalised Anime Mastodon instance which means you can talk about whatever. Other languages are allowed but we ask for Mostly English.

The instance is themed on Mob Psycho 100 but it is not pinned to just the topic of Anime, I just thought the domain would be cool and like the show üì∫.

Rules and such

Rules on Esper.lol are pretty light. The only time servers will be de-federated will be due to either being a bot-spammed mess or breaking our small list of rules on a wide scale. Please read the rules section below for a complete list of the server rules!

Moderated servers are only available to logged-in users due to blocked servers attempting to harass esper.lol after being blocked.

Why does this instance look different than vanilla Mastodon?

Esper.lol uses Bird-like UI as its theme. This instance is 100% vanilla Mastodon directly from the official git repo. If you are interested in the Bird-like UI theme the source code can be found here: Github

DMCA Takedown Requests

Only content posted by users with "@esper.lol" in their username is managed by this server. If you find infringing material from a user on our server, make sure it is not covered by Fair Use, then send your takedown request to this instance's admin email ([hello] @ [spiritsandsuch.lol]).
